# dice-game
dice-game

번갈아 가면서 주사위를 던지는 게임.
<br/> 
먼저 50점을 만들면 이긴다.
<br/> 
1~2는 플레이어의 현재 점수를 초기화하고 턴을 넘긴다.
<br/> 
3~6은 현재 점수에 나온 주사위 숫자를 더한다.
<br/> 
이후에 홀드 하고 현재 점수를 누적 점수에 더할지, 계속 주사위를 던질지 선택.



https://main--den-dice-game.netlify.app/


# 개선사항
rolldice 할 때 주사위 이미지가 변경되는 식인데 주사위 이미지 변경되는 것이 좀 느려서 보기에 불편하다.
